# Learnify — Online Learning Platform (Lab Assignment 5)

## Project theme
Online Learning Platform (Landing page with course cards, testimonials, pricing, and contact form).

## Files
- `index.html` — main page (semantic HTML5)
- `style.css` — CSS (Flexbox + Grid, variables, media queries)
- `README.md` — this file

## Features implemented
- Header with navigation and CTA buttons
- Hero banner with search CTA and animated entrance
- Popular categories list
- Featured courses displayed in CSS Grid (cards)
- Testimonials section
- Pricing plans
- Contact / sign-up form (labels + required attributes)
- Footer with social links

## Accessibility & best practices
- Semantic elements: `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`
- `alt` attributes on images
- `aria-label` on navigation, forms
- Skip-to-content link
- Responsive images (use appropriately sized images in `images/` folder)

## Breakpoints used
- Mobile: ≤ 768px — single-column layout, stacked header
- Tablet: ~768–1023px — two-column layout for course grid
- Desktop: ≥ 1024px — three-column course grid and full header

## How to run
1. Clone repository or download files.
2. Open `index.html` in any modern browser.
3. (Optional) Upload to GitHub Pages for live preview.
